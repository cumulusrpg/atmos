# atmos (JavaScript)

JavaScript implementation of the atmos event-sourcing engine.

The design lives at the repo root in [`../DESIGN.md`](../DESIGN.md). This package
targets that design. It should stay behaviorally equivalent to the Go
implementation at the repo root — parity is enforced by shared fixtures and
cross-runtime equivalence tests (TBD).

## Status

Empty scaffold. API surface not yet implemented.

## Running tests

```sh
npm test
```

Uses Node's built-in test runner (Node >= 18).
